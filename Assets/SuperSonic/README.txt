Supersonic UI Package Usage Guide - BETA

- Inside the folder Supersonic->HyperUI->Prefabs you can find prefabs for common UI menus and screens
- Note that currently the UI is mainly visual assets to use and manipulate as you please
- To use a UI screen, you must add a Canvas to your scene, and do the following:
	- Set UI Scale Mode to "Scale with Screen Size"
	- Set Reference Resolution to 1080x1920 (mobile portrait default resoultion)
	- Set Screen Match Mode to "Match Width or Height"
	- Set Match parameter to 0.5
- Now you can add any UI screen to your scene, under the Canvas you created.
